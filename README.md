# ACWeChat
基于 xmppframework, IM 后端使用 openfire，简单的 p2p 通讯，需要配置 openfire 即可使用。  
逐渐在维护项目，模仿微信部分功能。
  
  
# 功能  
- [x] 登录注册 (need openfire)  
- [x] 好友聊天，添加好友 (need openfire)  
- [x] 建立聊天室 (need openfire)
- [x] 朋友圈（界面) 
- [ ] 个人信息页面  
- [ ] 上传图片  
- [ ] 发动态
